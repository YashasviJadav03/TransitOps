# Sprint 1: Authentication and Core Data Models

## Objective
Establish secure access to the platform and define the foundational entities for users, vehicles, and drivers. 

## Key Deliverables
- **User Modeling:** Implement the User data model, including a mandatory Role field (e.g., Fleet Manager, Driver).
- **Secure Authentication:** Develop the sign-up and login workflows using email and password, backed by secure session or JWT handling.
- **Route Protection:** Implement security checks to ensure that only authenticated users can access the application's internal routes.
- **Core Entities:** Define the database models for Vehicles and Drivers, encompassing all mandatory fields specified in the project requirements.
