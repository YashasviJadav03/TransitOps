# Sprint 2: Resource Management

## Objective
Build the core management modules for vehicles and drivers, ensuring data integrity and providing a clean interface for record manipulation.

## Key Deliverables
- **Vehicle Registry Module:** Implement Create, Read, Update, and Delete (CRUD) operations for vehicles. The system must strictly enforce the uniqueness of the Vehicle Registration Number.
- **Driver Management Module:** Implement CRUD operations for driver profiles. Critical fields include license expiry tracking and operational status.
- **User Interface:** Develop foundational list and table interfaces incorporating status badges for quick visual reference.
